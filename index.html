API Documentation

You can use this app externally. Direct GET request to /api/rest will return full JSON docs.

Example cURL Upload:

curl -X POST "https://YOUR_SPACE_URL/api/rest" \
  -H "password: YOUR_PASSWORD" \
  -F "file=@yourfile.mp4" \
  -F "slug=myvideo" \
  -F "format=redirect"

Setting format=redirect triggers a 308 permanent redirect to the file instantly.
